# Objectives and Non-goals

## Objectives

- Keep a practical, explicit cross-platform build system for a small lab team
- Support Windows and Ubuntu targets
- Build VTK from source and install target-specific SDK + wheel outputs
- Operate Python-first orchestration with `pmanager`
- Keep the design extensible to future external libraries

## Non-goals

- Public PyPI release strategy
- CI/CD requirements
- Application feature development
- Production installer design
- Plugin architecture for external libraries
